アプリの開き方

  1. zipフォルダ「K Photobooth」を解凍します。
  2. 解凍すると「K Photobooth.app」アプリファイルが出てきます。
  3. 任意の場所にアプリ用フォルダを作成し、その中に「K Photobooth.app」を入れます。
  4. 「K Photobooth.app」を開きます。
  5. アプリを開くと、同じフォルダ内に「system」フォルダが自動生成されます。

起動時に生成されるsystemフォルダ内のフォルダ・ファイルについて

フォルダ構造（ツリー図） 
📂 アプリ用フォルダ（任意の名前）
│
├─📄 K Photobooth.app（アプリ本体）
└─📂 system（アプリ起動時に自動生成）
    │
    ├─📂 frames
    │   └─🖼️ [フレーム画像ファイルをここに追加]
    │      （※フレーム画像がない場合は自動でサンプルフレームが生成されます）
    ├─📂 secure_data
    │   └─🔒 [アプリの内部処理に使用される]
    ├─📂 sounds
    │   └─🔊 [効果音ファイル（mp3/wav）をここに追加・変更可能]
    │      
    └─📄 app_settings.json
  • framesフォルダ
    フレーム画像を入れるためのフォルダです。季節やイベントごとに好きな画像を追加できます。
    ここに入れた画像は、準備画面のフレーム選択で利用できます。
  • secure_dataフォルダ
    アプリの内部処理で使われるデータが保存されます。
  • soundsフォルダ
    撮影時やカウントダウンで鳴る効果音ファイルが保存されます。
    効果音を変更したい場合は、ここに音声ファイル（mp3またはwav）を追加できます。
    シャッター音の場合は「shutter」、カウントダウン音の場合は「countdown」とファイル名を付けてください。
  • app_settings.json
    準備画面で設定した内容（カメラ・フレーム・印刷・Wi-Fi・タイトルなど）が保存されます。

これらのフォルダやファイルは、アプリを起動したときに自動で作成されます。
フレーム画像や効果音を追加したい場合は、該当フォルダにファイルを入れてください。

画面構成

  • 準備画面：カメラ・フレーム・印刷・Wi-Fi・タイトル等の設定
  • 撮影画面：カメラ映像を表示し、撮影・カウントダウン・合成処理
  • フレーム選択画面：利用するフレームを選択
  • 結果表示画面：
    QRコード表示(Wi-FiQR設定またはHTTPSサーバーをオンにしている場合)
    再印刷ボタン(印刷をオンにしている場合表示)
    終了ボタン(撮影画面に戻る)

準備画面の設定項目

  1. タイトル
    スマホ等で画像を受け取る際ページに表示されるタイトルです。イベント名など、現場に合わせて設定してください。
  2. 撮影間隔
    撮影時の間隔（カウントダウン）を設定します。
  3. 開始遅延
    カウントダウンとカウントダウンの音声のタイミングがずれる場合に使用します。数値を上げるとカウントダウンの開始が音声より遅くなります。
  4. 画像反転
    撮影画像を左右・上下に反転できます。
  5. 音声テスト
    撮影・カウントダウンなどに鳴る効果音のテスト再生ができます。
  6. 権限確認
    カメラ・ストレージなどの権限が正しく付与されているか確認できます。権限が不足している場合はOSの設定を見直してください。
  7. 印刷設定
    印刷する/しないの切り替えを設定します。
    ※「自動印刷」をオンにした場合のみ、以下のプリンター関連設定が表示されます。
    • プリンター
      接続済みのプリンターを選択します。プリンターが認識されない場合は接続やドライバーを確認してください。
    • 印刷縮小率
      印刷時の画像縮小率（倍率）を設定できます。用紙サイズやレイアウトに合わせて調整してください。
    • 座標グリッドテスト印刷
      印刷位置や倍率の確認のため、グリッド付きテスト印刷ができます。実際の用紙でズレがないか確認しましょう。
  8. Wi-Fi QR設定
    会場のWi-Fi情報（SSID・パスワード）を入力すると、来場者向けのQRコードが表示されます。スマホでQRコードを読み取るだけでWi-Fiに接続できます。
    ※使用するネットワークは必ず管理者の許可を得て設定してください。不特定多数の来場者が接続するため、セキュリティに十分注意してください。
  9. HTTPSサーバー
    撮影した画像をWi-Fi経由でスマホなどで受け取る場合、この機能をオンにする必要があります。
    画像を受け取る端末（スマホ等）は、必ず本アプリが動作しているPCと同じネットワーク内に接続してください。
    ※起動時は毎回セキュリティのためオフになります。利用する場合は必ず毎度でオンにしてください。
  10. 操作方法
    オンにすると各画面で来場者がわかりやすいように操作方法が表示されます。
    (タッチパネルで利用する場合などはオフにしてください)

運用手順

  1. アプリ起動後、準備画面で各種設定を行う
  2. 「開始」ボタンで撮影画面へ移動　((推奨)この際にアプリ画面をフルスクリーンにする)
  3. 来場者が「撮影」ボタンを押すと、カウントダウン→撮影→フレーム合成→プレビュー表示まで自動進行
  4. 自動印刷をオンにしている場合は１枚自動印刷される。再度印刷する場合は再印刷を押す。
  5. 終了ボタンを押して次の来場者へ
  6. ※終了ボタンを押すと撮影した画像は削除されます。写真を受け取る前に押さないようご注意ください。

スマホで写真を受け取る際の注意事項

  • セキュリティのため、写真受け渡しページはHTTPS通信を利用しています。
  • そのため、スマホでアクセスすると「警告」や「安全ではありません」などの表示が出る場合がありますが、これは本アプリの仕様です。
【Chromeでアクセスする場合】
警告画面が表示されたら「詳細設定」→「192.168.x.xxにアクセスする(安全ではありません)」を選択してください。
【Safariでアクセスする場合】
警告画面が表示されたら「詳細を表示」→「このWebサイトを閲覧」を選択してください。
※警告は本アプリのセキュリティ仕様によるものです。安心してご利用ください。

フレームの作成方法について

自動生成されるサンプルフレームを利用してカスタムフレームを作成してください。
サンプルフレームを編集することで、簡単にオリジナルフレームが作れます。

  • フレーム画像は PNG形式(透過) で作成してください。
  • フレーム画像の「撮影画像が入る部分」は透明（透過）になっており、そこに撮影した写真が合成されます。
  • 推奨アプリ：アイビスペイント などを使ってオリジナルフレームを作成できます。
  • 撮影画像が入る部分にもデザインを入れることが可能です。
  • 撮影画像が入る部分に半透明の色や模様を入れると、写真にフィルター効果を加えることもできます。

作成したらframesフォルダに入れてください。
